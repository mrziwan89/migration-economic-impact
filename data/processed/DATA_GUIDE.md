## 1. `eurostat_master_panel.csv` ‚Äî The Source of Truth

**Generated by:** `notebooks/test.ipynb` ‚Üí Cell 20  
**Rows:** ~9,747 | **Columns:** 5

### Schema

| Column | Type | Description | Example |
|--------|------|-------------|---------|
| `Year` | int | Reference year | `2023` |
| `Country` | str | ISO 3166-1 alpha-2 country code | `DE`, `FR`, `PL` |
| `Cohort` | str | Population group (see below) | `Extra_EU_Migrant` |
| `Value` | float | Metric value (units depend on Metric) | `23.1` |
| `Metric` | str | Which pillar indicator (see below) | `Temporary_Contract_Rate` |

### `Cohort` Values

| Code | Meaning | Eurostat Source Code |
|------|---------|---------------------|
| `Native` | Born in reporting country | `NAT` |
| `EU_Migrant` | Born in another EU-27 member state | `EU27_2020_FOR` |
| `Extra_EU_Migrant` | Born outside the EU-27 | `NEU27_2020_FOR` |
| `Total_Migrant` | All foreign-born combined (used for Incarceration only) | `FOR` |

> **Note:** `EU_Migrant` and `Extra_EU_Migrant` use 2020 EU definitions. Pre-2020 data is mapped from `EU27_2007_FOR` / `NEU27_2007_FOR`.

### `Metric` Values

| Metric | Pillar | Unit | Source Dataset |
|--------|--------|------|----------------|
| `Highly_Educated_Employment_Rate` | 1 ‚Äî Brain Waste | % employed | `lfsa_ergaedn.csv` |
| `Temporary_Contract_Rate` | 2 ‚Äî Precarious | % on temp contracts | `lfsa_etpgan.csv` |
| `Incarceration_Rate_Per_100k` | 3 ‚Äî Incarceration | Rate per 100k inhabitants | `crim_pris_ctz.csv` |
| `Poverty_Rate` | 4 ‚Äî Poverty Risk | % at risk of poverty | `ilc_peps05.csv` + `estat_ilc_peps05n_en.csv` |

### How to Use

```python
import pandas as pd

df = pd.read_csv('data/processed/eurostat_master_panel.csv')

# Get brain waste employment gap for Germany
df_de = df[(df['Country'] == 'DE') &
           (df['Metric'] == 'Highly_Educated_Employment_Rate')]

# Pivot cohorts into columns for direct comparison
df_pivot = df_de.pivot_table(
    index=['Year', 'Country'],
    columns='Cohort',
    values='Value'
).reset_index()
# Now df_pivot has: Native | EU_Migrant | Extra_EU_Migrant columns
```

## 2. `eurostat_insights.csv` ‚Äî Disparity Gap Table

**Generated by:** `notebooks/test.ipynb` ‚Üí Cell 30  
**Rows:** ~3,125 | **Columns:** 8  
**This is the primary file for visualization cells.**

### Schema

| Column | Type | Description | Example |
|--------|------|-------------|---------|
| `Year` | int | Reference year | `2023` |
| `Country` | str | ISO 3166-1 alpha-2 | `ES` |
| `EU_Migrant` | float | Value for intra-EU migrants | `16.3` |
| `Extra_EU_Migrant` | float | Value for non-EU migrants | `38.7` |
| `Native` | float | Value for native-born | `14.2` |
| `Gap_ExtraEU_vs_Native_PP` | float | `Extra_EU_Migrant ‚àí Native` (percentage points) | `24.5` |
| `Metric` | str | Pillar indicator | `Temporary_Contract_Rate` |
| `Ratio_Foreign_vs_Native` | float | `Foreign / Native` ratio (Incarceration only) | `2.08` |

### Reading the Gap Column

The `Gap_ExtraEU_vs_Native_PP` column means different things per metric:

| Metric | Positive gap means... | Negative gap means... |
|--------|----------------------|----------------------|
| `Highly_Educated_Employment_Rate` | Migrants employed MORE (rare) | ‚öÝÔ∏è Migrants employed LESS ‚Üí brain waste |
| `Temporary_Contract_Rate` | ‚öÝÔ∏è Migrants stuck in MORE temp jobs | Migrants have stable jobs |
| `Poverty_Rate` | ‚öÝÔ∏è Migrants face MORE poverty | Migrants face less poverty |
| `Incarceration_Rate_Per_100k` | ‚öÝÔ∏è Foreigners over-represented in prisons | Foreigners under-represented |

> **Rule of thumb:** A negative gap for Employment = bad for migrants. A positive gap for Poverty/Temp/Incarceration = bad for migrants.

### Pillar-Specific Notes

#### Pillar 1 ‚Äî Brain Waste (`Highly_Educated_Employment_Rate`)
- **Filter:** `age=Y20-64`, `isced11=ED5-8` (tertiary educated), `sex=T`
- **Interpretation:** Employment rate of highly-educated people. Lower = more brain waste.
- **Coverage:** 40 countries, 1995‚Äì2025

#### Pillar 2 ‚Äî Precarious Employment (`Temporary_Contract_Rate`)
- **Filter:** `age=Y20-64`, `sex=T`
- **Interpretation:** % of employed people on temporary contracts. Higher = more precarity.
- **Coverage:** 40 countries, 1995‚Äì2025

#### Pillar 3 ‚Äî Incarceration (`Incarceration_Rate_Per_100k`)
- **Special columns:**
  - `Extra_EU_Migrant` = `Foreign_Share_Pct` (% of prison population that is foreign)
  - `Native` = `Native_Share_Pct` (% of prison population that is national)
  - `Ratio_Foreign_vs_Native` = `FOR_count / NAT_count` (e.g. `2.1` = foreigners 2.1√ó over-represented)
- **Coverage:** 37 countries, 2008‚Äì2024 (Germany `DE` not in source data)
- ‚öÝÔ∏è `EU_Migrant` column is always empty for this metric ‚Äî the source only distinguishes `FOR` (all foreign) vs `NAT` (national)

#### Pillar 4 ‚Äî Poverty Risk (`Poverty_Rate`)
- **Filter:** `age=Y18-64` or `Y_GE18`, `sex=T`
- **Eurostat transition:** Pre-2021 data from `ilc_peps05.csv`, 2021+ from `estat_ilc_peps05n_en.csv`
- ‚öÝÔ∏è **Known unit issue:** The new dataset (`estat_ilc_peps05n_en.csv`) returns values in **thousands of persons** for some countries in 2021+, not percentages. Filter to `Year <= 2020` for clean percentage comparisons, or use the master panel directly.

### How to Use

```python
import pandas as pd

df = pd.read_csv('data/processed/eurostat_insights.csv')

# ‚îÄ‚îÄ Brain Waste choropleth data (2014-2024 EU countries only)
brain_waste = df[
    (df['Metric'] == 'Highly_Educated_Employment_Rate') &
    (df['Year'].between(2014, 2024)) &
    (df['Country'].str.len() == 2)   # Exclude EU/EA aggregates
].copy()

# ‚îÄ‚îÄ Incarceration ratio per country (latest year)
incarceration = df[
    df['Metric'] == 'Incarceration_Rate_Per_100k'
].dropna(subset=['Ratio_Foreign_vs_Native'])

latest_inc = incarceration.sort_values('Year').groupby('Country').last().reset_index()
# Ratio_Foreign_vs_Native > 1 = foreigners over-represented
```

## 3. `brain_waste_by_region.csv` ‚Äî US IPUMS Regional Data

**Generated by:** `notebooks/test.ipynb` ‚Üí Cell 11  
**Rows:** 1,122 | **Columns:** 7  
**Source:** IPUMS ACS 2014‚Äì2023

### Schema

| Column | Type | Description | Example |
|--------|------|-------------|---------|
| `YEAR` | int | Survey year | `2019` |
| `STATEFIP` | int | FIPS state code | `6` (California) |
| `US_State` | str | State name | `California` |
| `is_imigrant` | bool | `True` = foreign-born, `False` = US-born | `True` |
| `total_educated_pop` | float | Weighted count of highly-educated people | `4500000.0` |
| `brain_waste_pop` | float | Weighted count in low-skill jobs | `1200000.0` |
| `brain_waste_pct` | float | `brain_waste_pop / total_educated_pop √ó 100` | `26.7` |

> **Note:** `is_imigrant` has a typo (one 'm') ‚Äî this is the original column name from the notebook. Keep it consistent in any new code.

### How to Use

```python
import pandas as pd

df = pd.read_csv('data/processed/brain_waste_by_region.csv')

# Compare immigrant vs native brain waste in California
ca = df[df['US_State'] == 'California']
pivot = ca.pivot_table(index='YEAR', columns='is_imigrant', values='brain_waste_pct')
# pivot[True]  = immigrant brain waste %
# pivot[False] = native-born brain waste %
```

## 5. Known Caveats & Gotchas

| Issue | Affects | Detail |
|-------|---------|--------|
| **Poverty unit mismatch post-2021** | `eurostat_insights.csv` Poverty rows | `estat_ilc_peps05n_en.csv` returns thousands of persons for some countries, not %. Use `Year <= 2020` for clean % comparisons. |
| **Germany missing from incarceration** | `eurostat_insights.csv` Incarceration | `crim_pris_ctz.csv` does not include Germany. Expected ‚Äî Eurostat does not report this for DE. |
| **`EL` = Greece (not `GR`)** | All Eurostat files | Eurostat uses `EL` for Greece. Map to `GRC` for Plotly. |
| **`EU_Migrant` empty for incarceration** | `eurostat_insights.csv` | Prison data only distinguishes Foreign vs National ‚Äî no EU/non-EU split. Use `Ratio_Foreign_vs_Native` instead. |
| **`is_imigrant` typo** | `brain_waste_by_region.csv` | Single 'm' ‚Äî matches original IPUMS notebook output. Do not rename without updating all downstream code. |
| **Eurostat aggregates** | `eurostat_insights.csv` | Rows with `Country` = `EU`, `EA`, `EU27_2020` are EU-wide averages. Exclude them when mapping to country-level choropleths. |
